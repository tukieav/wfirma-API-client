## Opis

Mikroserwis do integracji z API wFirma i obsługi webhooków.

## Instalacja

npm install

## Uruchomienie

npm start

## Testy

npm test


## Konfiguracja

Utwórz plik `.env` z następującymi zmiennymi:

PORT= 

WFIRMA_APP_KEY=

WFIRMA_ACCESS_KEY=

WFIRMA_SECRET_KEY=

WFIRMA_API_URL=https://api2.wfirma.pl

COMPANY_ID=

WEBHOOK_KEY=

DB_USER=

DB_PASSWORD=

DB_HOST=

DB_PORT=

DB_DATABASE=
